Esta aplicación fue realizada en React con TypeScriptt, Graphql y CSS.
### ¿Cómo se configura?

## FRONTEND  
1. Crear folder vacío
2. clonar el repositorio
4. git flow init
6. Rama develop
5. navegar hasta la carpeta socialnetwork
6. npm install
6. npm start
## BACKEND 

1. navegar hasta la carpeta src/graphql
2. npm istall
3. node index.js
